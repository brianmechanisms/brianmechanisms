Universal Linear Motion Wheel Mechanisms

Introduction

A universal linear motion wheel(ULMW) is a type of mechanism used to create compound linear motion mechanisms to enable long range linear motion. Practical ULMWs can be classified according to: 1. Type of motion 2. Number of links 3. Number of Degrees of Freedom 4. Type of locomotion scheme

Classification according to the type of motion

  1. Continuous motion
  2. Intermittent motion

Classification according to the number of links

  1. 5 Bar ULMW
  2. 6 Bar ULMW

Classification according to the number of DoFs

  1. 1 DoF
  2. 2 DoF

Classification according to the number of DoFs

  1. Wheeled
  2. Legged

Technically wheeled ULMWs can be converted to legged ULMWs by adding legs to them.

Several other types of ULMW mechanisms can be built.
